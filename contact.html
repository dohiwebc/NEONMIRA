所属レーベル

NEON MIRA は CHROME SIGNAL RECORDS 所属です。

CHROME SIGNAL RECORDS は、サイバー美学とシティポップを軸にした近未来レーベル。音楽だけでなく、ビジュアル・ステージ・デジタル体験まで一貫した世界観を構築する。出演依頼、コラボ、メディア取材、ビジュアル制作などのお問い合わせは、下記メールまたはフォームよりご連絡ください。

連絡先 info@chromesignal.example

通常2〜3営業日以内にご返信いたします。お急ぎの場合は件名に【急】とご記載ください。

お問い合わせ内容
  • 出演・ライブ（Booking）
  • コラボレーション
  • メディア・取材
  • ビジュアル制作
  • ライセンス・使用許諾

FAQ

よくある質問
// FAQ_01 QUERY

ライブ出演の依頼はどこに送ればいいですか？

A:出演・ライブに関するご依頼は、info@chromesignal.example または下記フォームの「出演・ライブ」を選択してお送りください。日程・会場・規模などをご記載いただけるとスムーズです。

// FAQ_02 QUERY

楽曲の使用許諾（CM・映像など）は可能ですか？

A:使用目的・媒体・期間などを明記のうえ、フォームまたはメールでお問い合わせください。レーベル側で審査のうえ、ご返答いたします。

// FAQ_03 QUERY

ファンレター・プレゼントは受け付けていますか？

A:現時点では公式SNSのメッセージ機能、またはフォームよりお気持ちをお送りください。物品の送付先については個別にご案内いたします。

// FAQ_04 QUERY

返信までどのくらいかかりますか？

A:通常2〜3営業日以内にご返信します。内容によってはお時間をいただく場合があります。お急ぎの場合は件名に【急】とご記載ください。
